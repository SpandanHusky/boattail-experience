🚘 BoatTail Experience
A clean and precise front-end clone of the Rolls-Royce Boat Tail website, focused on pixel-perfect UI and smooth animations. Built using HTML, CSS, and JavaScript to replicate a refined, high-end digital experience.

✨ Overview
This project recreates the elegant and minimal design language of a luxury automotive website, with a strong focus on visual detail, layout accuracy, and seamless interaction.

🎯 Features
(1) Pixel-perfect UI replication
(2) Smooth and subtle animations
(3) Clean and minimal design
(4) High attention to spacing, typography, and flow

🛠️ Tech Stack
HTML
CSS
JavaScript

⚠️ Disclaimer
This project is a front-end clone created for educational purposes only.
It is not affiliated with or endorsed by Rolls-Royce.
All design and brand assets belong to their respective owners.

⚙️ Note on Assets
A high-resolution video asset has been optimized/removed to maintain repository efficiency. The visual experience remains consistent with the original design intent.
